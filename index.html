Genetic Genealogy Tips
Shared DNA %
Parent/child: ~50% · Full sibling: ~50% · Half-sibling: ~25% · 1st cousin: ~12.5% · 2nd cousin: ~3.1%
Endogamy Caution
Communities with high intermarriage (Ashkenazi Jewish, Acadian, colonial American) show inflated DNA matches. Use segment data.
Chromosome Browsers
Use 23andMe, MyHeritage, or GEDmatch chromosome browsers to identify which specific segments are shared with matches.
Upload Raw DNA
Upload raw DNA to GEDmatch, MyHeritage, or FamilyTreeDNA for cross-platform matching across all major databases.

GitHub Gist Sync
How Gist sync works
Your archive is saved as a secret GitHub Gist — a private text file in your GitHub account. Only you can see it (not publicly listed). On any device where you open FamilyArchive Pro and enter the same token + Gist ID, your full archive loads automatically. Syncing is one button-press: push to save, pull to load.
1
GitHub Personal Access Token
Create one at github.com/settings/tokens → click Generate new token (classic) → check only the gist scope → copy and paste here. This token never leaves your device.
Must be a classic token. GitHub's newer "fine-grained" tokens (often the default option now) cannot create Gists at all — using one will make every sync fail.
2
Gist ID (leave blank to create a new one on first push)
After your first push, the Gist ID is saved here automatically. Copy it to other devices.
3
Auto-sync options
Conflict resolution
A conflict happens when both local and remote data have changed since the last sync.
Ancestry.com Integration
Ancestry.com Integration
Direct API sync with Ancestry.com is not available in this version — Ancestry does not offer a public API for third-party apps. The best way to move data between Ancestry and FamilyArchive Pro is via GEDCOM export.
Import from Ancestry
In Ancestry: open your tree → Settings → Export Tree → Download .ged file. Then use the Import button (⬆ in the header) to load it here.
Export to Ancestry
Use Reports → GEDCOM (.ged) to export your archive, then import that file into Ancestry via your tree's Member Connect or Import feature.
Use the Import button to bring in a GEDCOM export from Ancestry.
Import Family Data
GEDCOM is exported by Ancestry.com, FamilySearch, RootsMagic, Legacy, and most genealogy software. To export from Ancestry: Tree Settings → Export Tree → Download.
